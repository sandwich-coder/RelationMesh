> [!NOTE]
This is a public snapshot of my private repository of paper *RelationMesh* in progress.
As an extension of my previous paper *RelationMesh: 설명 가능한 비지도 이상탐지 모델* (CISC-W'25),
all the Pandas pipelines have been replaced by the Polars to read and write bigger datasets faster.
Also, **Processor** class has been added to trim high-cardinality features and handle out-of-vocabulary in a more systematic way.
